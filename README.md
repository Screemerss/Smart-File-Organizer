# Smart File Organizer

A simple but powerful Python application to automatically organize files in a folder based on their extension. Keep your "Downloads" folder clean forever!


## ‚ú® Features

*   **Automatic Sorting**: Monitors a folder and moves files into subdirectories (`Images`, `Documents`, `Video`, etc.).
*   **Easy to Use**: Simple graphical interface built with Tkinter.
*   **Customizable**: Easily extend the file types by modifying the `FILE_TYPES` dictionary in the script.
*   **Safe**: Runs in the background without blocking the user interface.

## üöÄ How to Use

1.  **Prerequisites**: You need to have Python 3 installed on your system.
2.  Download the `smart_organizer.py` script from this repository.
3.  Run the script from your terminal:
    ```bash
    python smart_organizer.py
    ```
4.  Click **"Browse..."** to select the folder you want to organize (e.g., your Downloads folder).
5.  Click **"Start Organizing"**.
6.  That's it! The bot will now monitor the folder and move new files automatically.
